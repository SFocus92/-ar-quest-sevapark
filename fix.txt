
Значит где-то всё же пытаешься грузить MindAR как обычный скрипт, но в файлах что ты дал — загрузки MindAR нет. То есть либо она в другом файле (может `11k2689yki-cw.js` или `0yk_bz6ihtacv.js` которые ты не скинул), либо в `public/` лежит какой-то старый HTML с тегом `<script>`.

**Что нужно исправить:**

### 1. В `layout.tsx` (строка с meta):

### 2. В `page.tsx` добавить загрузку MindAR правильным способом

В функции `loadARLibs()` после загрузки A-Frame добавить загрузку MindAR **UMD-версии**:

```typescript
async function loadARLibs(): Promise<boolean> {
  if ((window as any).AFRAME && (window as any).MINDAR) {
    return true;
  }
  
  return new Promise((resolve) => {
    const loadScript = (src: string, isModule = false) => new Promise<void>((res) => {
      if (document.querySelector(`script[src="${src}"]`)) {
        res();
        return;
      }
      const s = document.createElement('script');
      s.src = src;
      s.async = true;
      s.crossOrigin = 'anonymous';
      if (isModule) s.type = 'module';
      s.onload = () => res();
      s.onerror = () => res();
      document.head.appendChild(s);
    });
    
    // Грузим A-Frame
    loadScript('https://aframe.io/releases/1.4.0/aframe.min.js')
      .then(() => {
        // Ждём загрузки THREE
        return new Promise<void>(res => {
          const check = setInterval(() => {
            if ((window as any).THREE) {
              clearInterval(check);
              res();
            }
          }, 100);
        });
      })
      .then(() => {
        // Важно: используем UMD версию MindAR (без import/export)
        return loadScript(
          'https://cdn.jsdelivr.net/npm/mind-ar@1.2.5/dist/mindar-image-three.umd.prod.js'
        );
      })
      .then(() => setTimeout(() => resolve(true), 500))
      .catch(() => resolve(false));
  });
}
```
