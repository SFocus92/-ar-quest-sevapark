# AR-Квест "СеваПарк" 🗺️🎮

Интерактивный AR-квест для парков развлечений на базе WebAR с распознаванием реальных объектов.

## ✨ Особенности

- ✅ Работает в браузере (не требует установки приложения)
- ✅ Поддержка iOS Safari и Android Chrome
- ✅ NFT-маркеры (распознавание реальных объектов через камеру)
- ✅ 7 последовательных этапов квеста
- ✅ 3D-объекты и анимации
- ✅ Финальная награда с промокодом **SEVA2024AR**

## 🚀 Быстрый старт

## 🎮 Как играть

1. Откройте ссылку на мобильном устройстве
2. Разрешите доступ к камере
3. Наведите камеру на объекты в парке
4. Проходите этапы по порядку (1→2→3→4→5→6→7)
5. Получите промокод в финале!

## 🔧 Настройка NFT-маркеров

### Шаг 1: Оптимизация фотографий (уже сделано)

```bash
node scripts/optimize-photos.js
```

Результат: `public/assets/nft-sources/marker-1.jpg` до `marker-7.jpg`

### Шаг 2: Генерация NFT-дескрипторов

1. Откройте https://carnaux.github.io/NFT-Marker-Creator/
2. Загрузите каждый `marker-X.jpg` из `public/assets/nft-sources/`
3. Нажмите "Generate" и скачайте 3 файла: `.fset`, `.fset3`, `.iset`
4. Переименуйте их в `marker-X.fset`, `marker-X.fset3`, `marker-X.iset`
5. Поместите все файлы в `public/assets/nft/`

**Важно:** Без NFT-дескрипторов AR-распознавание работать не будет. Используйте tap-to-simulate для тестирования логики.

## ⚙️ Кастомизация

### Изменение промокода

Отредактируйте `src/lib/quest-config.ts`:

```typescript
export const PARK_CONFIG = {
  name: "СеваПарк",
  promoCode: "ВАШ_КОД_2024",
  discount: "20%",
};
```

### Изменение текстов этапов

В `src/lib/quest-config.ts` измените массив `STEPS`:

```typescript
{
  id: 'marker_gate',
  order: 1,
  title: 'Новое название',
  description: 'Новое описание',
  hint: 'Новая подсказка',
  // ...
}
```

## 📱 Требования

- Node.js 18+
- HTTPS для продакшена (критично для камеры!)
- Современный браузер с WebRTC
- iOS Safari 11+ или Android Chrome 56+

## 🐛 Устранение неполадок

### Камера не работает
- Убедитесь что используется HTTPS (не HTTP)
- На iOS используйте Safari (не Chrome)
- Разрешите доступ к камере в настройках браузера

### Маркеры не распознаются
- Убедитесь что NFT-дескрипторы созданы и размещены в `public/assets/nft/`
- Проверьте освещение (должно быть хорошим)
- Держите камеру на расстоянии 30-100 см от объекта

### Тестирование без маркеров
В режиме разработки тапните по экрану для имитации обнаружения маркера (7 секций экрана = 7 маркеров).
